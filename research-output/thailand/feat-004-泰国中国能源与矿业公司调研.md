# 泰国中国能源与矿业公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search (4组查询)

---

## 摘要

中国能源企业在泰国以光伏和数字能源为主力。华为数字能源与Chow Energy合作, 2026年前开发超100MW太阳能+储能项目, 累计支持超3,000个全国项目。中国东方电气承建乌隆塔纳大坝水上光伏项目(2024年3月商运), 为东南亚最大水上光伏之一。美的与AIS、中国联通、华为联合打造东南亚首个5G全连接工厂。泰国目标2030年碳达峰、2050年碳中和、2065年净零排放。

## 一、华为数字能源

### PEA合作

| 维度 | 详情 |
|------|------|
| 合作方 | 省级电力局(PEA) |
| 内容 | 供应屋顶光伏逆变器和优化器 |
| 覆盖 | 12个分区人员培训 |

### Chow Energy联盟

| 维度 | 详情 |
|------|------|
| 合作始于 | 2014年 |
| 开发目标 | 2026年前超**100MW**太阳能+储能 |
| 累计项目 | 支持Chow Energy超**3,000个**全国项目 |

### 住宅光伏方案

华为"1+4+X"方案:
- 智能充电 + 逆变器 + 储能 + 管理 + 显示
