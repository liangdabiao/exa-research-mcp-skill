# 泰国中国基建公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search (4组查询)

---

## 摘要

中泰高铁是"一带一路"在东南亚的旗舰项目, 曼谷-廊开全线610公里总投资约5,208亿泰铢(~153亿美元)。一期(曼谷-呵叻253公里)完成51.74%, 目标2030年运营, 但比原计划落后约9年。二期(呵叻-廊开357公里)已于2025年2月获批, 2025年招标。中老铁路已通车(2021年), 累计运送旅客超3,000万人次, 为中泰铁路南延提供物流基础。

## 一、中泰高铁(核心项目)

### 项目总览

| 阶段 | 路线 | 里程 | 投资额 | 完成进度 | 运营目标 |
|------|------|------|--------|---------|---------|
| 一期 | 曼谷-呵叻 | 253公里 | 1,794亿泰铢(~53亿美元) | **51.74%** | 2030年 |
| 二期 | 呵叻-廊开 | 357公里 | 3,414亿泰铢(~100亿美元) | 已获批, 2025年招标 | 2031年 |
| 合计 | 曼谷-廊开 | **610公里** | **~5,208亿泰铢(~153亿美元)** | -- | -- |

|------|------|

| 轨道系统合同 | 1个 |

- Bang Sue-Don Mueang段: CP集团争议

| 维度 | 详情 |
|------|------|
| 车站 | 5座(Bua Yai、Ban Phai、Khon Kaen、Udon Thani、Nong Khai) |

| 铁路线路建设 | 3,357亿泰铢 |
| 货运中心 | 57亿泰铢 |

### 主要中资承包商

| 企业 | 角色 |
|------|------|
| **中国铁建(CRCC)** | 中铁11局、23局负责一期土建, 合同额约4.15亿美元 |
| **中国中铁(CREC)** | 参与部分土建合同 |
| **中国中车(CRRC)** | 提供复兴号CR300AF动车组(250km/h) |
| **中国中建(CSCEC)** | 承建关键路段 |

### 合作模式
- 泰国政府全额出资承担土建和投资
- 中国提供技术、设计、隧道/桥梁施工, 并负责前3年运营
- 第3-7年联合运营
