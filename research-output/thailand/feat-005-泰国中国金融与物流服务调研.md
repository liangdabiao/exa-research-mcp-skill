# 泰国中国金融与物流服务深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search (4组查询)

---

## 摘要

2025年8月中泰签署新一轮本币互换协议, 规模**700亿元人民币/3,700亿泰铢**(~98.1亿美元)。但泰国对华贸易逆差持续恶化, 2025年1-11月达**606.4亿美元**(2.02万亿泰铢), 远超2024年全年的453.6亿美元。中国占泰国FDI申请近40%, 注册企业29,913家。COSCO收购林查班港股权深化东南亚布局, 中老铁路+林班-湄公快线实现跨境试运, 时效提升70%。

---

## 一、金融合作

### 本币互换

| 维度 | 详情 |
|------|------|
| 签署时间 | 2025年8月18日 |
| 规模 | **700亿人民币/3,700亿泰铢**(~98.1亿美元) |
| 有效期 | 5年, 自2025年12月22日起 |
| 签署方 | 中国人民银行(潘功胜) vs 泰国银行(赛塔普) |
| 目的 | 促进双边贸易投资本币结算 |

### 中资银行布局

| 银行 | 泰国业务 |
|------|---------|
| **ICBC(工商银行)** | 曼谷分行, 长期服务中泰经贸, 配合本币互换机制 |
| **BOC(中国银行)** | 曼谷设立机构 |
| **CCB(建设银行)** | 曼谷设立机构 |

### 在泰中资企业规模

| 指标 | 数据 |
|------|------|
| 注册企业数(2024.10) | **29,913家** |
| 注册资本总额(2024.10) | **4,093亿泰铢** |
| 中国占泰国FDI申请(2025.1-9) | 近**40%** |
| 中国在泰工作许可(2025.5) | **47,128人**(首超日本) |
| 中国注册企业资本(2025.5) | **4,545.5亿泰铢**(占外资10.3%) |
| 中国留学生(2025学年) | **23,243人**(最大留学生群体) |
