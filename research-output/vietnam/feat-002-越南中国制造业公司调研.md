# 越南中国制造业公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search
**分析框架**: feat-002 制造业领域

---

## 摘要

越南是中国制造业外迁的最大受益者之一。"China+1"战略驱动下，富士康累计投资超32亿美元、雇员6万+，立讯精密投资5.2亿美元扩至6-8万员工。越南北部工业区(北宁/北江/海防)与中国供应链深度耦合。Apple供应链持续将游戏机、VR、可穿戴等新品类产能转移至越南。

## 二、Apple供应链转移

| 转移品类 | 越南布局 |
|---------|---------|
| iPhone组装 | 富士康北宁(部分产能) |
| AirPods | 立讯精密 |
| iPad | 富士康/立讯 |
| VR/MR设备 | 立讯精密(Meta Quest) |
| 游戏机 | 富士康(PS5等) |
| 可穿戴设备 | 持续增加 |

**趋势**: Apple供应链从"在中国制造"转向"中国设计+越南制造"，新品类优先考虑越南。

## 三、产业链转移趋势

### China+1战略加速
| 指标 | 数据 |
|------|------|
| 中国外资排名 | 越南第六大外资来源国 |
| 新增项目数 | 2023年暴增**七倍** |

### 投资结构升级
| 转型方向 | 说明 |
|---------|------|
| 从 → 到 | 传统劳动密集型 → 电子/半导体/新能源/智能制造 |
| 重点领域 | 高技术制造、技术转让、循环经济、绿色能源 |
| 工业园区 | 胡志明市2025年吸引53亿美元投资, 中国为第二大外资来源 |

### 北部工业集群
