# 越南中国金融与物流服务公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search
**分析框架**: feat-005 金融与物流服务领域

---

## 摘要

工商银行(ICBC)深度嵌入越南金融体系：跨境QR支付互联互通已落地，人民币结算通道打通，承诺为胡志明市/岘港国际金融中心和基建项目融资。物流方面，海防港2025年首破200万TEU，COSCO全球网络支撑越南-欧洲航线，中越陆海新通道将重庆至越南从6天缩短至2天。2025年中越双边贸易额达2,960亿美元。

---

## 第一部分: 金融

### 一、ICBC(工商银行)在越南

| 时间 | 事件 |
|------|------|
| 2025.02 | 越南国家银行副行长与ICBC高级副总裁会晤, 讨论跨境支付合作 |
| 2025.04 | 中越签署跨境二维码支付互联互通协议 (ICBC+银联+NAPAS+越南外贸银行) |
| 2025.12 | 越南正式启动与中国跨境QR支付连接 |
| 2026.03 | ICBC行长拜会越南财政部, 承诺为国际金融中心融资, 支持铁路基建项目 |

### 二、人民币合作

| 维度 | 数据 |
|------|------|
| 2025年中国跨境人民币收付 | **71万亿元** |
| 人民币结算占比 | 占中国跨境收付超**50%** |
| ICBC+Vietcombank | 已提供本币跨境结算服务 |
| 银联卡 | 2005年进入越南, Lazada/越南航空等已接受 |

### 三、金融合作重点
- 跨境QR支付互通 (已落地)
- 人民币清算与外汇掉期
- 出口信贷融资 (机械设备出口)
- 国际金融中心建设 (胡志明市、岘港)

## 第二部分: 物流与航运

- 运营从胡志明市/头顿出发的多条航线
- 覆盖北欧、地中海、中东及大洋洲, 主要经**新加坡**中转
- 2025年COSCO全球吞吐量**1.53亿TEU** (+6.2%)
