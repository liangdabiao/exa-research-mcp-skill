# 巴西中国科技与互联网公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search
**分析框架**: feat-003 科技与互联网领域

---

## 摘要

中国在巴西科技领域的影响力体现在三大维度: AI合作(中巴联合AI实验室/葡语大模型)、5G(华为为关键供应商, 巴西拒绝美国封杀)、互联网平台(TikTok成Z世代文化现象, Shein/Temu/AliExpress渗透电商)。ByteDance计划在巴西建设300MW数据中心, 与ZTE联合开发AI手机。

---

## 一、AI与数字合作

### 中巴AI合作框架 (2025年5月卢拉访华)

| 维度 | 详情 |
|------|------|
| 协议 | AI合作MOU, 为期3年, 自动续签 |
| 覆盖领域 | 基础设施、培训交流、应用平台、AI安全、人才培养 |
| 联合AI实验室 | 训练**葡语大语言模型**(华为开发基础版LLM) |
| 首批行业 | 政务、医疗、教育、安全、农业 |
| 巴西AI投资计划 | 至2028年投入超**47亿美元** |

### Huawei在巴西

| 维度 | 详情 |
|------|------|
| Dataprev合作 | 技术转让协议, 建设AI中心, 服务东北部大学 |
| 华为云 | 与Edge UOL谈判推出网络安全和AI服务 |
| 企业网络 | 2025年8月圣保罗峰会, 900+行业代表 |
| 大客户 | 巴西银行Bradesco等 |

### 5G博弈
- 美国持续施压排除华为5G, 提供EXIM银行融资替代方案
- 巴西坚持"战略不结盟", **拒绝封杀中国科技公司**

### BRICS数字合作
- 巴西2025年金砖轮值主席国, AI治理和数字包容为议程核心
- 新开发银行(NDB)支持AI可持续发展项目
- CBERS卫星项目为地学智能和气候监测提供合作基础

## 二、互联网平台

### TikTok/ByteDance

| 维度 | 详情 |
|------|------|

| 数据中心 | 计划在塞阿拉州建设**300MW**数据中心(与Casa dos Ventos合作) |

### 中国电商

| 平台 | 巴西市场 |
|------|---------|
| Shein | 2024年跨境购物平台使用率**35.7%** |
| AliExpress | 使用率从44%降至41% (受新关税影响) |
| Temu | 2025年新进入, 扩张激进 |

### 政策影响
- 2024年8月起, 巴西对低价进口商品征收**20%关税**
- 多州ICMS从17%提至20%
- 中国跨境电商平台增长放缓
