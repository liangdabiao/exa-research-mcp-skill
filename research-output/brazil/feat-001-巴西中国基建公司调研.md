## 摘要

中资企业在巴西基建领域的旗舰项目为**两洋铁路**(Bioceanic Railway), 全长约3,000公里, 投资35亿美元, 连接大西洋与太平洋。2025年卢拉访华期间与中国中铁签署MOU正式启动联合可行性研究。国家电网为巴西最大输电运营商之一, 控制24家输电特许经营商, 累计承诺投资397亿美元。COSCO投资13亿美元建设秘鲁钱凯港, 天津至巴西航程从54天缩短至40天。

---

## 一、两洋铁路 (Bioceanic Railway) — 南美世纪工程

| 维度 | 详情 |
|------|------|
| 总长度 | 约3,000公里 |
| 投资 | 约35亿美元 (约180亿雷亚尔) |
| 路线 | 巴西巴伊亚州伊列乌斯港 → 秘鲁钱凯港 |
| 整合 | FIOL + FICO + 南北铁路 |
| 预计完工 | 5-8年 |
| 战略意义 | 缩短大西洋-太平洋航运距离10,000+公里, 挑战巴拿马运河 |

**时间线**:
| 时间 | 事件 |
|------|------|
| 2025年4月 | 中国中铁代表团赴巴西实地考察 |
| 2025年5月 | 卢拉访华, 巴西规划部长与中国中铁签署MOU |
| 2025年7月 | Infra SA与中国铁路经济规划研究院正式签约 |
| 待定 | 美国视此为地缘战略扩张, 巴西坚持"战略不结盟" |
