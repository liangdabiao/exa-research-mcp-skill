# 印尼中国金融与物流服务深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search

---

## 摘要

中印尼金融合作在2025年5月升级至新高度: 本币结算(LCT)扩展至**所有国际收支项目**, 本币互换扩容至**4,000亿元人民币**(约548亿美元)。工银印尼2025年上半年LCT交易量增长超200%。双边贸易额1,352亿美元, 中国为印尼第一大贸易伙伴, 但印尼对华贸易逆差达**205亿美元**为最大单一国逆差来源。

## 三、双边贸易

### 核心数据

| 指标 | 数据 |
|------|------|
| 2024年双边贸易额 | **1,352亿美元** |
| 中国为印尼第一大贸易伙伴 | 连续多年 |
| 印尼对华贸易逆差(2025) | **205亿美元** (最大单一国逆差) |
| 逆差同比(1-7月) | +84.8% |

### 贸易结构失衡

| 方向 | 主要商品 |
|------|---------|
| 印尼对华出口 | 矿物燃料(27%)、钢铁(72%)、棕榈油(15%)、镍(75%)、纸浆(47%) |
| 中国对印尼出口 | 电气机械(85)、机械设备(84%)、有机化学品(29%)、塑料(39%) |

### 印尼国会政策建议
1. 加速下游化: 镍基电池、棕榈油化工品
2. 出口市场多元化: 善用IA-CEPA等FTA
3. 最大化FTA利用率
4. 对华贸易外交: 争取加工品准入
