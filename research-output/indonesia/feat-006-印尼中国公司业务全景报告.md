# 印尼中国公司业务全景报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search (3组合并搜索, 12组查询)

---

## 核心发现

印尼是中国海外投资最深、风险与机遇并存的东南亚国家。CATL 60亿美元Project Dragon是全球最大海外电池投资, 中国控制印尼75%镍冶炼产能。雅万高铁73亿美元从标杆变为"金融定时炸弹", 教训深刻。本币结算框架扩展至所有国际收支, 互换规模4,000亿元(增60%)。但印尼对华贸易逆差达205亿美元, 结构性矛盾突出。数据中心容量将从900MW扩张至1.41GW, Batam挑战新加坡。

---

## 一、行业全景

### 1. 基建 — 高铁教训
- 雅万高铁73亿美元, 债务重组中
- Batang园区22亿美元投资快速推进 ("印尼的深圳")
- 水电/地热/光伏全面布局

### 2. 制造业 — 镍+手机
- 中国控制**75%**镍冶炼产能
- CATL 60亿美元全产业链投资
- 中国手机品牌占**70%**市场份额

### 3. 科技 — 数据中心爆发
- 容量900MW → 1.41GW (2029)
- TikTok 1.65亿用户(全球最大)
- 华为云利用率91%

### 4. 能源 — 可再生转型
- 能源投资56%流向可再生能源
- 中国电建Cirata 192MW浮式光伏(东南亚最大)
- 全球镍供过于求, LFP替代趋势

### 5. 金融 — 脱美元加速
- LCT扩展至**所有**国际收支项目
- 本币互换**4,000亿人民币** (增60%)
- ICBC LCT交易量增长超200%

### 6. 物流 — 航线优化
- COSCO SE1新航线(印-马-印尼-新)
- 丹戎不碌港为中印尼贸易枢纽

```
印尼 (镍矿) → 中国 (冶炼/电池) → 越南 (组装/消费)

```

## 三、风险矩阵

| 风险 | 严重度 | 印尼特点 |
|------|--------|---------|
| **贸易逆差** | 高 | 205亿美元, 结构性矛盾突出 |
| **镍供过于求** | 高 | LFP替代, 镍价跌至成本线 |

| **地缘竞争** | 中 | 中美在关键矿产竞争加剧 |

## 四、战略建议

### 对中资企业
1. **镍产业**: 警惕LFP替代风险, 向下游(电池/正极材料)延伸以提升附加值
2. **数据中心**: 抓住Batam低成本优势, 挑战新加坡枢纽地位
3. **TikTok**: 维持Tokopedia合规运营, 关注反垄断审查
4. **金融**: 利用LCT和本币互换降低汇率风险
5. **贸易**: 主动推动印尼出口能力建设, 缓解逆差政治压力

### 对政策制定者
1. **下游化加速**: 推动镍/棕榈油/钢铁下游加工, 提升出口附加值
2. **出口多元化**: 善用FTA降低对中国单一市场依赖
3. **数据中心**: 支持Batam基础设施, 但避免监管过度宽松
4. **债务管理**: 吸取雅万高铁教训, 大型基建需透明融资
5. **AI布局**: 利用NVIDIA合作加速主权AI发展, 培养本地人才
