# 印尼中国基础设施建设公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search

---

## 摘要

雅万高铁(Whoosh)是中国"一带一路"在印尼的旗舰项目, 全长142km, 投资73亿美元, 2023年10月通车。但客运量远低于预期, 成为"金融定时炸弹"。Batang工业园区("印尼的深圳")快速发展, 2025年11月签约16个项目21.9亿美元。中国企业在印尼基建广泛布局, 涵盖镍冶炼、燃煤电厂、水电站、收费公路等。

---

## 一、雅万高铁 (Whoosh)

| 维度 | 详情 |
|------|------|
| 长度 | 142公里 |
| 设计时速 | 350km/h |
| 投资 | **73亿美元** (从60.7亿膨胀) |
| 通车 | 2023年10月 |
| 融资 | 中国国家开发银行75%贷款 (利率2%) |
| 股权 | 印尼国企联盟60% + 中国铁路国际40% |

**运营困境**:
- 2024年客运量约510万, 远低于预期
- 2025年上半年仅290万乘客
- PSBI 2024年亏损约4.2万亿印尼盾 (2.53亿美元)
- 2026年4月Danantara主权基金宣布债务重组方案
- 印尼政府决定动用国家预算偿还约70亿美元债务

**延伸计划**: 普拉博沃确认研究雅加达-泗水(780km)高铁可行性
