# 马来西亚中国房地产与能源公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search
**分析框架**: feat-004 房地产与能源领域

---

## 摘要

中资房地产企业在马来西亚经历从扩张到收缩的显著转变。碧桂园Forest City仅入住约2万人(远低于70万目标)，万科搁置吉隆坡项目，雅居乐亏本出售。但政策出现转机——Forest City纳入柔新特区并获免税区地位。能源领域则呈现强劲增长，中广核成为第二大独立发电商(5.08GW装机)，隆基绿能投资RM54亿建成东南亚最完整光伏制造基地，国家电网与TNB签署电网现代化合作。

---

## 第一部分: 房地产

### 一、Forest City (碧桂园)

#### 现状
| 维度 | 数据 |
|------|------|
| 实际入住 | 约2万人 (2025年) |
| 规划目标 | 70万人 (2035年) |
| 完成进度 | 4座人工岛中完成1座 |
| 已建公寓 | 约30栋 (26,000套), 大量空置 |
| 租金参考 | RM2,000/月以下 (两室双卫) |

#### 碧桂园债务危机
- 总负债约**1,640亿美元** (2023年末)
- 2023年净亏损1,780亿人民币, 2024年亏损328亿人民币
- 市值较2018年峰值跌去97%
- 中国国内超3,000个未完工项目
- 与债权人进行离岸债务重组谈判, 若失败面临清算

#### 政策转机
| 政策 | 内容 |
|------|------|
| 柔新特区 (SEZ) | 2025年1月签署协议, 享受贸易与商业便利 |
| 免税区地位 | 2024年获批, 支持旅游和本地经济 |
| 特殊金融区 | 家族办公室0%税率 (马标准24%), 最低RM3,000万 |
| MM2H签证放松 | 固定存款从RM100万降至US$150,000 |

### 二、其他中资房地产项目

#### 万科 (Vanke) — 吉隆坡 Jalan Raja Chulan
- 2017年以RM5亿购入7.4英亩地块
- 原规划RM50亿综合开发 (含80层塔楼), 因公众反对审批未通过
- 2023年"探索替代方案": 重新规划/合资/出售
- 财务相对稳健 (现金143亿美元, 覆盖2.2倍短期债务), 但海外扩张明显放缓

#### 绿地集团 (Greenland Group)
| 项目 | 投资 | 详情 |
|------|------|------|
| Danga Bay Jade Palace | RM6亿 | 13.6英亩, 2,000套住宅, 80%马国买家 |
| Tebrau Bay 水滨城 | RM24亿 | 128英亩, 5期15年, 最终20,000套住宅 |
| 总承诺 | RM200亿 | 自称"中国在马单笔最大房地产投资" |

#### 雅居乐 (Agile Group) — Embassy Garden
- 2023年11月以RM3.1亿出售, **亏损RM3.71亿**
- 地块前身为英国大使馆, 11年内三次易手
- 折射中资开发商在马资产减值趋势

#### 富力地产 (R&F) — Princess Cove
- 新山Causeway旁混合开发, 约3,200套住宅, 涉填海工程

### 三、政策与监管变化

| 变化 | 影响 |
|------|------|
| 外国人购房门槛提高 | 从RM500,000提高至RM1,000,000 |
| 环境审批趋严 | Forest City规划面积削减30% |
| 中国资本外流管控 | 2017年实施, 中国海外房地产投资从400亿降至100亿美元峰值 |
| 政治转向 | 马哈蒂尔2018年曾威胁禁购 → 安华政府2024年积极支持 |

### 四、中资开发商退潮趋势
- 恒大、碧桂园相继违约
- 雅居乐亏本出售, 万科搁置项目
- Savills: "中国开发商普遍趋向减少海外投资和财务义务"

## 第二部分: 能源

### 一、中广核 (CGN) — 马来西亚第二大独立发电商

| 维度 | 数据 |
|------|------|
| 总装机容量 | **5.08 GW** |
| 收购交易 | 2015年以RM98.3亿从1MDB收购Edra 13个清洁能源资产 |
| 累计供电 | 超2,000亿kWh |

**核心项目**:
| 项目 | 详情 |
|------|------|
| Edra Melaka燃气电站 (EMPP) | 装机2.24GW, 马最大燃气电站, 供应半岛约11%电力, 效率超60% |
| Bestari浮体光伏电站 | 2026年2月开工, 300MW, 覆盖433公顷废弃矿湖, 年发电6.4亿kWh, 供15万户 |
| 吉打光伏电站 | 105.2公顷, 2019年运营, 供电约3万户, 附带农光互补(菠萝种植) |

### 二、隆基绿能 (LONGi) — 光伏制造巨头

| 维度 | 数据 |
|------|------|
| 在马累计投资 | **RM54亿** (约83亿人民币) |
| 员工总数 | 超8,000人 (99.9%为马来西亚本地人) |
| 亚太总部 | 吉隆坡 |

**制造布局**:
| 基地 | 详情 |
|------|------|
| 砂拉越古晋 | 2016年建厂, 首个中国境外基地, 全球最完整单晶硅垂直一体化 (硅棒-硅片-电池-组件) |
| 雪兰莪Serendah | 2023年10月投产, 140英亩, RM18亿, 总产能8.8 GW/年, 雇员约2,000人 |

### 四、电网合作

**国家电网 (SGCC) x Tenaga Nasional Berhad (TNB)**:
- 2025年10月签署MoU
- 三大合作领域: 电力系统规划与并网、智能弹性电网、数字化与AI电网管理
- 支撑东盟电网互联 (ASEAN Power Grid) 愿景

### 五、投资环境

| 指标 | 数据 |
|------|------|
| 2035年可再生能源目标 | 装机占比40% |
| 2025年可再生能源目标 | 装机占比31% |
| 光伏制造优惠政策 | 最高100%所得税减免, 全额免税10-15年 |
| 马来西亚光伏模块产能 | 约50 GW (东南亚最大中资份额) |
