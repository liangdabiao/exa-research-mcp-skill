## 摘要

马来西亚是中国制造业出海东南亚的核心枢纽之一，尤其在半导体、新能源、钢铁和汽车制造领域。受China+1战略和中美贸易摩擦驱动，中资制造企业加速向马来西亚转移产能。槟城"东方硅谷"聚集超80家中资企业，马中关丹产业园累计投资超400亿元人民币。比亚迪、吉利/Proton、闻泰科技等头部企业在马布局CKD组装和半导体封测。

## 二、半导体/电子制造

### 1. Nexperia (闻泰科技控股)
- 计划投资超**3亿美元**在马新建工厂
- 目标2026年中将**约90%产能**从中国转移至马来西亚
- 受中美贸易摩擦及荷兰政府干预影响

### 2. Jingsheng Mechanical & Electrical (JSG)
- 子公司SuperSic在槟城Bertam Technology Park建厂
- 年产**24万片8英寸碳化硅晶圆**

### 3. 槟城半导体生态
| 指标 | 数据 |
|------|------|
| 跨国公司数 | 350+家 |
| 中资企业数 | 80+家 |
| 全球封测市场份额 | 13% |
| 全球IC出口排名 | 第6位 |
| 2025年前三季度FDI同比 | +22% |

**战略定位**: 中资企业从"关税规避"转向以槟城为**全球运营基地**
- 策略: "China for China, Penang for the rest of the world"
- FMM槟城分会: 中国AI/自动化技术可帮助马来西亚产业升级

## 三、钢铁与重工业

### 马中关丹产业园 (MCKIP) — "两国双园"模式

| 维度 | 数据 |
|------|------|
| 总投资 | 超400亿元人民币 (约58亿美元) |
| 累计工业产值 | 超1000亿元人民币 |
| 入驻项目 | 15个 |
| 涵盖行业 | 钢铁、轮胎、造纸、汽车零部件、食品加工 |

**旗舰项目 — 联合钢铁 (Alliance Steel)**:
- 年产能超**350万吨**
- 采用中国最先进技术
- 将马来西亚从钢铁**进口国**转变为**出口国**
- 雇员4,000+, 薪资高于当地平均水平

**关丹港协同**: 北部湾港集团入股后, 日吞吐量从1万吨提升至3万吨, 建成2个15万吨级深水泊位

## 四、汽车零部件

### Trensor (无锡天瑞)
- 在槟城设立首个海外工厂, 占地1万平方米
- 年产**1,000万只压力传感器**
- 供应当地车企Perodua (市占率超40%)
- 创造200-300个本地就业岗位

## 六、产业链转移趋势

| 趋势 | 说明 |
|------|------|
| China+1驱动 | 地缘政治压力加速中企向马来西亚转移产能 |
| 半导体聚焦 | 尤其集中在封测、碳化硅等高附加值环节 |
| 新能源集群 | 电动车CKD组装+电池配套形成产业集群 |
| 从出口到运营 | 策略从"规避关税"升级为"全球运营基地" |
| 技术输出 | 中国AI/自动化技术带动马来西亚产业升级 |

## 七、宏观背景

- 2024年中马贸易额**2,120亿美元**, 中国连续16年第一大贸易伙伴
- 槟城2025年前三季度制造业FDI同比+22%, 中国为第二大投资来源国
