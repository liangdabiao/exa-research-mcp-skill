# 新加坡中国房地产与能源公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search
**分析框架**: feat-004 房地产与能源领域

---

## 摘要

与马来西亚的大规模房地产投资（如Forest City）不同，中资企业在新加坡房地产领域以高净值个人投资为主，企业级开发相对克制。能源领域则呈现强劲增长：中国通过跨境清洁能源项目积极参与新加坡能源供应，中核集团(CNNC)的400MW太阳能+储能项目、宁德时代(CATL)的2.2GWh储能供应，标志中资企业在新加坡能源转型中扮演关键角色。

---

## 第一部分: 房地产

### 一、投资特点

| 特征 | 说明 |
|------|------|
| 投资主体 | 中国高净值个人 > 中资企业开发商 |
| 投资类型 | 商业楼宇、住宅 > 大型开发项目 |
| 咨询服务 | Prestige Property(北京总部,新加坡分支)专门面向大中华区客户 |
| 监管环境 | ABSD额外买方印花税调控外国需求, 但未阻止中国买家 |
| 对比马来西亚 | 新加坡以投资为主, 马来西亚以开发为主 |

### 二、相关企业

| 企业 | 业务 | 备注 |
|------|------|------|
| Prestige Property | 新加坡房产投资顾问(北京总部) | 反映中国高净值人群对新加坡的持续兴趣 |
| Ying Li International | 光大集团旗下, 高端商业/住宅开发 | 核心在重庆, 非新加坡主阵地 |
| Haleton Group | 华资企业, 商业地产开发/资产管理 | 2006年成立, 新港中三地业务 |

### 三、与马来西亚对比

| 维度 | 新加坡 | 马来西亚 |
|------|--------|---------|
| 中国开发商活跃度 | 低(无大规模开发项目) | 高(Forest City/绿地等) |
| 投资驱动力 | 资产避险+财富管理 | 产业布局+China+1 |
| 政策环境 | ABSD冷却措施, 成熟监管 | MM2H放松, 特区优惠 |
| 趋势 | 稳健, 高净值投资持续 | 房地产退潮, 能源科技兴起 |

## 第二部分: 能源

### 一、跨境清洁能源项目 (核心)

#### 中核集团(CNNC) + Equator Renewables Asia — 印尼廖内群岛

| 维度 | 详情 |
|------|------|
| 规模 | 900MW太阳能 + 1.2GWh电池储能(BESS) |
| 输新容量 | **400MW** (AC) 通过海底电缆输往新加坡 |
| 中方角色 | CRE International(CNNC子公司): 发电侧投资/建设/运营 |
| 新方角色 | ERA: 传输和购电协调 |
| 预计完工 | 2029年 |
| 意义 | EMA授予六张有条件清洁电力进口许可证之一 |

#### Vanda太阳能+电池项目

| 维度 | 详情 |
|------|------|
| 规模 | 2GWp光伏 + 4.4GWh BESS (印尼廖内群岛) |
| 输新容量 | EMA有条件批准300MW出口至新加坡 |
| CATL储能 | 宁德时代供应**2.2GWh** EnerX BESS |
| 光伏组件 | 隆基1GW + 天合光能1.4GW |
| 本地化 | 三家中国制造商均在印尼建厂满足TKDN要求 |

| 企业 | 在项目中的角色 |

### 三、新加坡清洁能源目标

| 目标 | 数据 |
|------|------|

### 四、全球清洁能源投资

| 维度 | 数据 |
|------|------|
| 2024-25中国海外清洁能源投资 | **800亿美元** |

| 全球前6大电池供应商 | **全部为中国企业** (CATL/海辰/亿纬/比亚迪/中创新航/瑞浦) |

### 五、区域电网合作

- 东盟电网(ASEAN Power Grid)计划2045年连接成员国
- AIIB联合区域银行拟动员**60亿美元**用于基础设施
- 柔南可再生能源走廊(SJREC): 4GW+5.12GWh, 获IFC 60亿美元投资
- 马来西亚-新加坡氢气进口: 2029年启动, 首期年减碳1.8万吨CO2

## 六、战略判断

1. **新加坡房地产**: 中资以个人投资为主, 企业级开发远不如马来西亚活跃
2. **能源合作是亮点**: 中核/CATL/LONGi等通过跨境项目深度参与新加坡能源转型
3. **中国供应链优势**: 全球前6大电池供应商均为中国企业, 在东南亚清洁能源中占据主导
4. **政策驱动**: 新加坡6GW清洁电力进口目标为中国企业创造巨大市场
