# 新加坡中国制造业公司深度调研报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search
**分析框架**: feat-002 制造业领域

---

## 摘要

新加坡中资制造企业以半导体供应链服务和精密工程为主，核心制造基地仍在中国。但2025年中国对新加坡固定资产投资承诺从2.5%飙升至20.6%，首次超越美国，主要驱动力为半导体扩产潮（服务AI需求）。中资企业更多以销售窗口、设备服务和供应链上游环节参与新加坡制造生态。

---

## 一、宏观投资趋势

| 指标 | 数据 |
|------|------|
| 2025年新加坡固定资产投资承诺 | S$142亿 (制造业占85%) |
| 中国占比 | **20.6%** (2024年仅2.5%) |
| 中国企业商业支出占比 | **50.7%** (2024年仅15%) |
| 新加坡半导体研发投资 | S$8亿 (2026预算) |
| 驱动因素 | 中美地缘政治、AI芯片需求、离岸总部需求 |

## 二、主要中资/关联制造企业

### 1. GENORI Singapore (臻宝科技)
- 母公司: 重庆臻宝科技
- 业务: 先进陶瓷、石英、碳化硅消耗品, 面向8寸/12寸晶圆厂
- 雇员: 新加坡2人(销售团队)
- 认证: ISO 9001/14001/45001

### 2. ASETS (Advanced Semiconductor Equipment Technology Singapore)
- 2024年注册, 半导体设备销售/组装/维修
- 核心团队来自中微半导体(AMEC)
- 雇员: 11人(年增长200%)
- 覆盖: 新加坡、德国、日本、美国

### 3. Global TechSolutions (GTS)
- 前道半导体设备集成、翻新和维修
- 客户: TSMC、GlobalFoundries、Micron、China Semicon
- 雇员: 9人
- 服务: CVD、扩散、蚀刻、光刻、PVD全流程

## 五、中国企业新加坡投资动因

1. **地缘政治避险**: 中美博弈背景下新加坡的"中立"定位
2. **离岸总部**: 大量中企在新加坡注册区域总部(如MiniMax、蚂蚁国际)
3. **AI需求**: 半导体扩产潮带动设备/服务需求
4. **资金/人才池**: 利用新加坡资本市场和国际化人才
