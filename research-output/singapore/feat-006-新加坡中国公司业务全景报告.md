# 新加坡中国公司业务全景报告

**调研日期**: 2026年4月10日
**数据来源**: Exa Advanced Web Search (8组并行搜索, 90+条结果)
**分析方法**: 6大行业领域系统性调研 + 跨行业趋势综合分析 + 新马对比

---

## 核心发现

2025年是中国对新加坡投资的历史性转折年：固定资产投资承诺占比从2.5%飙升至**20.6%**，首次超越美国(17.3%)；中国企业占新加坡总商业支出的**50.7%**。这一剧变的核心驱动力是中美地缘政治博弈——大量中企将新加坡作为离岸总部和区域枢纽。与马来西亚的大规模实体投资不同，中企在新加坡更多以总部经济、金融服务、科技研发和跨境能源供应为切入点。

## 一、行业全景分布

### 1. 基础设施建设 — 稳健深耕

**代表企业**: CSCEC(中建)、CCECC/CREC(中铁)、CHEC(中港湾)

| 核心指标 | 数据 |
|---------|------|
| CSCEC累计HDB组屋 | 50,241套 (每20个新加坡人有1人住CSCEC建造的房子) |
| CSCEC全球排名 | ENR全球承包商**第1** |
| CREC突破 | 首次进入新加坡地铁供电+盾构隧道 |
| 2025年中国投资占比 | **20.6%** (首次超越美国) |

**关键趋势**: 从传统建筑商向设施运营转型, 轨道交通领域实现零的突破

### 2. 制造业 — 服务导向型布局

**代表企业**: GENORI(臻宝)、ASETS、GTS, 间接参与Micron/UMC/Vanguard扩产

| 核心指标 | 数据 |
|---------|------|
| 中资制造形态 | 销售窗口+设备服务为主, 核心制造仍在中国 |
| Micron累计投资 | US$600亿, 全球98%闪存在新加坡生产 |
| UMC扩产 | US$50亿, 22/28nm新加坡最先进代工 |
| 新加坡半导体研发 | S$8亿(2026预算) |

**关键趋势**: 中企以供应链上游(陶瓷/石英耗材)和设备服务参与, 大规模制造以台资/美资为主

### 3. 科技与互联网 — 总部经济爆发

**代表企业**: ByteDance、Ant International、MiniMax、Whale、eLightCloud

| 核心指标 | 数据 |
|---------|------|
| ByteDance营收 | 1,550亿美元 (+29%), 净利润330亿 |
| 蚂蚁国际总部 | 新加坡, 1,198人, 20亿笔跨境交易 |
| 数据中心空置率 | **1.4%** (亚太最低) |
| AI独角兽迁址 | Manus迁新加坡, 估值5亿→20亿(半年) |
| 华为5G | 被排除主网络, 仅获TPG区域合同 |

**关键趋势**: 总部迁址潮, 数据中心供不应求, 华为5G受阻但AI突围

### 4. 能源 — 跨境清洁能源供应

**代表企业**: CNNC(中核)、CATL(宁德时代)、LONGi(隆基)、Trinasolar

| 核心指标 | 数据 |
|---------|------|
| CNNC输新项目 | 400MW太阳能+1.2GWh储能(2029年) |
| CATL储能供应 | 2.2GWh EnerX BESS |
| 新加坡清洁电力进口目标 | 2035年6GW (约占1/3) |
| 中国海外清洁能源投资 | 800亿美元 (2024-25) |
| 全球前6大电池供应商 | 全部为中国企业 |

**关键趋势**: 中国通过跨境项目深度参与新加坡能源转型, 储能供应链主导地位

### 5. 金融 — 区域金融中心核心

**代表企业**: ICBC、BOC、CCB

| 核心指标 | 数据 |
|---------|------|
| 人民币清算 | ICBC 2013年(首家境外清算行) |
| 清算格局 | 2025年DBS获批第二家, 形成"双清算行" |
| ICBC贸易融资 | ICBC集团境外机构**第一** |
| 中新ETF通 | 10只ETF, AUM同比+42.5% |
| CCI项目 | 323项协议, 总金额257.5亿美元 |

**关键趋势**: 人民币国际化加速, 资本市场互联互通深化

### 6. 物流与航运 — 双枢纽布局

**代表企业**: COSCO(中远海运)

| 核心指标 | 数据 |
|---------|------|
| 裕廊岛枢纽一期 | 87,500m², 3个月达90%+使用率 |
| 裕廊岛枢纽二期 | 1.42亿美元, 2026年完工 |

## 二、新加坡 vs 马来西亚对比

| 维度 | 新加坡 | 马来西亚 |
|------|--------|---------|
| **投资性质** | 总部经济/金融/研发 | 实体投资/制造/基建 |
| **基建角色** | 建筑承包商 | 一带一路旗舰(ECRL) |
| **制造业** | 销售窗口+设备服务 | 大规模工厂(电动车/半导体/钢铁) |
| **科技** | 全球总部迁址 | 数据中心集群(柔佛溢出) |
| **5G** | 华为被排除 | 华为获U Mobile第二网络 |
| **房地产** | 高净值个人投资 | 大型开发(Forest City/绿地) |
| **能源** | 跨境清洁电力进口 | 本地发电+光伏制造 |
| **金融** | 境外人民币清算中心 | 贸易融资+基建支撑 |
| **物流** | 综合供应链枢纽 | 港口运营+跨境电商 |
| **中资定位** | 离岸总部/资金池/避风港 | 产能基地/市场腹地/资源 |

### 互补关系

新加坡和马来西亚在中资企业东南亚布局中形成明显的**功能互补**：

```
新加坡 (大脑): 离岸总部 → 资金管理 → 技术研发 → 品牌运营
     ↕ 人才/资本/数据流动
马来西亚 (四肢): 制造基地 → 基建项目 → 市场腹地 → 资源供应
```

典型案例：
- ByteDance: 新加坡总部 + 柔佛数据中心
- COSCO: 新加坡裕廊岛枢纽 + 巴生港运营
- LONGi: 新加坡亚太总部 + 雪兰莪/砂拉越制造基地
- 蚂蚁: 新加坡全球总部 + 马来西亚Alipay+拓展
- 中核: 新加坡清洁电力需求 + 印尼/马来西亚项目供应

## 三、跨行业趋势

### 1. 投资爆发式增长
- 2025年中国投资占比从2.5%→20.6%, 商业支出从15%→50.7%
- 地缘政治是核心驱动力, 非基本面变化

### 2. 总部经济集群
- ByteDance、Ant International、MiniMax、Whale、eLightCloud均设新加坡总部
- Manus案例: 迁址新加坡后估值4倍增长

### 3. 数据中心溢出效应
- 新加坡空置率1.4%, 柔佛州承接溢出(2029年容量预测5,000 MW)
- ByteDance、GDS中标新加坡首批产能, 同时布局柔佛

### 4. 人民币国际化加速
- ICBC新加坡清算行 + DBS第二清算行
- e-CNY数字人民币试点
- 中新ETF通、A股第二上市

### 5. 清洁能源跨境供应
- 中国全球前6大电池供应商主导供应链
- 跨境太阳能+储能成为新加坡能源转型关键路径

## 四、风险矩阵

| 风险 | 严重度 | 新加坡特点 |
|------|--------|-----------|
| **地缘政治** | 高 | 中美博弈的核心战场, 中企被迫"选边"压力 |
| **监管合规** | 中-高 | ABSD/反洗钱/数据安全等成熟但复杂的监管体系 |
| **数据中心瓶颈** | 中 | 1.4%空置率限制增长, 新产能释放需时间 |
| **运营成本** | 中 | 全球最高运营成本之一, 人才/租金/电力 |
| **华为5G受限** | 低-中(已兑现) | 主网络已排除华为, 边缘化态势难逆转 |
| **过度集中风险** | 中 | 中国企业商业支出占比50.7%, 政策变化影响大 |

## 五、战略建议

### 对中资企业
1. **双城战略**: 新加坡(总部/金融/研发) + 马来西亚(制造/基建/市场)协同布局
2. **合规优先**: 新加坡监管成熟严厉, 反洗钱/数据安全/税务合规成本高但不可省
3. **数据中心**: 关注新加坡新产能分配, 同时布局柔佛作为成本替代
4. **能源机会**: 6GW清洁电力进口目标为储能/光伏企业创造巨大市场
5. **品牌建设**: 利用新加坡国际品牌效应反哺全球业务(如Manus案例)

### 对政策制定者
1. **平衡术**: 中美博弈中维持"中立"定位是新加坡最大优势
2. **数据中心**: 加速新产能释放, 同时引导溢出到柔佛等合作伙伴
3. **人才竞争**: 与中企合作培养本地AI/半导体人才, 留住外流人才
4. **清洁能源**: 加速跨境电力项目审批, 2035年6GW目标需大量新增项目
5. **金融深化**: 利用双清算行格局进一步扩大人民币业务
